Our Dataset Website On Video Game Sellings With Ratings.

Dataset Overview For this project, we used a video games dataset acquired from Kaggle. The dataset contains approximately up to 17000 rows varying from video games released between 1980 and 2016. Each row contains an individual video game, and includes details such as their name, developer, genre, and their global sales figures. Furthermore, the dataset provides sales data for different regions, such as North America, Japan, Europe and other regions. In addition, it contains review related variables such as critic score, user score, critic count, and user count, which allows for a deep analysis of the relationship between a video game’s quality and its sales performance. The dataset was chosen as it meets the project requirements by meeting over 200 rows and several numerical and categorical variables. It is also suitable for the machine learning section, as it provides an ample amount of data for training models. Data Cleaning Summary Before beginning the data analysis, the dataset had to be cleaned to improve data quality. A copy of the original dataset was made in excel and labelled raw_data to have the original data. Non-numerical values such as “tbd” in the user_score column and “N/A” in the year of release were replaced with blank slots. The values in user_score were treated as missing data so that Excel could carry out its calculations correctly for the descriptive statisitics. Missing values in columns such as critic_score, were treated as genueine gaps in the data rather than errors.
